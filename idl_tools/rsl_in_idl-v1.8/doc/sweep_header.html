Sweep Header Structure

Member Type Description
field_type string A string of two characters identifying the type of radar data collected in this sweep, for example, 'DZ'.
sweep_num int Number of sweep within volume scan (1 to nsweeps).
fixed_angle float Fixed angle of antenna in degrees. This is the nominal elevation angle for PPI, azimuth angle for RHI.
beam_width float Ray beam width in degrees.
vert_half_bw float Vertical half beam width.
horz_half_bw float Horizontal half beam width.
nrays int Number of rays in sweep.
