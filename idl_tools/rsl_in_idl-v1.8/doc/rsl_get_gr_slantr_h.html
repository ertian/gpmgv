rsl_get_gr_slantr_h

Get ground range, slant range, and height, given a ray and a bin index.

Syntax

rsl_get_gr_slantr_h, ray, bin_index, ground_range, slant_range, height

Arguments

Input
ray A ray structure.
bin_index Index of the range bin, where 0 is the first.
Output
ground_range Ground range to radar target, in kilometers.
slant_range Range to radar target, in kilometers.
height Height in kilometers.

Keywords

None
