Volume Header Structure

Member Type Description
field_type string A string of two characters identifying the type of radar data in this scan, for example, 'DZ'. See field types below.
nsweeps long Number of sweeps in volume scan.
calibr_const float Calibration constant.
no_data_flag float Missing or deleted data flag. Usual value is -32767.
